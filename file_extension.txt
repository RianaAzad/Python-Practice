Use .py:
For regular Python code, scripts, and modules.
When you are writing any Python code that needs to be executed or imported.
Use .pyc and .pyo:
No need to manually create these; they are automatically generated by Python.
Use .pyw:
When creating a Windows GUI application to avoid the console window.
Use .pyd:
When creating a compiled extension module for Python on Windows.
Use .pxd, .pxi, and .pyx:
When working with Cython to write C extensions for Python.
Use .pyi:
When providing type hints for static type checking in Python modules.
